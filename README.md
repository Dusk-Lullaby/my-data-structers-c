# Data Structures and Algorithms

本项目记录学习数据结构使用C语言实现的代码

## 开发环境

* **IDE**: Visual Studio 2022
* **构建系统**: MSBuild (基于 `.sln` 和 `.vcxproj` 结构)
* **版本控制**: Git

## 项目目录

* `my_SeqTable`：顺序表
* `my_LinkList`：单链表
* `my_doubleLoopLink`：双向循环链表
* `my_Stack`：栈
* `my_Queue`：队列
* `my_Joseph`：约瑟夫环问题实现
* `my_BinaryTreeTraversal`：二叉树的基础遍历（前序、中序、后序、层序）
* `my_BinarySearchTree`：二叉搜索树 (BST)
* `my_AVLTree`：AVL 平衡二叉树
* `my_ThrededBinaryTree`：线索二叉树
* `my_HuffmanTree`：哈夫曼树与编码
* `my_minHeap`：最小堆（优先队列）
* `my_UnionQuickSet`：并查集
* `my_MatrixGraph`：邻接矩阵
* `my_Adjacency`：邻接表
* `my_crossLinkGraph`：十字链表（用于有向图）
* `my_Prim`：普里姆算法 (Prim)
* `my_kruskal`：克鲁斯卡尔算法 (Kruskal)
* `my_Dijkstra`：迪杰斯特拉算法 (Dijkstra)
* `my_TopologicalSort`：拓扑排序
* `my_keyPath`：关键路径 (AOV/AOE 网)
* `my_insertSort`：插入排序
* `my_swapSort`：交换类排序（冒泡排序、快速排序等）
* `my_mergeSort`：归并排序
* `my_BinarySearch`：二分查找
* `my_RedBlackTree`：红黑树
